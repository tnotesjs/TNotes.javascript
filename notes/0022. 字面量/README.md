## 2. 📒 字面量

- 字面量在 JavaScript 中表示值。
- 字面量是固定的值，不是变量。
- 你可以在脚本中直接书写字面量。
- 字面量可以是各种数据类型，包括数值、字符串、布尔值、对象、数组等。

## 3. 💻 demos.1 - 不同类型的字面量

```javascript
// 你在程序中，直接把值给写出来，你写出来的值其实就是字面量。

// 以下是数字字面量
123
111

// 以下是字符串字面量
'hello'
'word!'

// 以下是布尔字面量
true
false

// 以下是数组字面量
[1, 2, 3]
['a', 'b', 'c']
```
