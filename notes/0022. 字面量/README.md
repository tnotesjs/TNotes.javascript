## 2. 🫧 评价

- 字面量（Literal）这个词可能很多人没有听过，其实字面量的概念很简单，字面量就是指那些直接写在代码中的值。

## 3. 📒 字面量（Literal）

- 字面量在 JavaScript 中表示值。
- 字面量是固定的值，不是变量。
- 你可以在脚本中直接书写字面量。
- 字面量可以是各种数据类型，包括数值、字符串、布尔值、对象、数组等。
