# JavaScript 权威学习资料

"最权威"取决于你的阶段和目标，以下按层级梳理：

---

## 一、语言规范（终极权威）

- **ECMAScript 语言规范 (ECMA-262)** https://tc39.es/ecma262/ 这是 JavaScript 的"法律文本"，由 TC39 委员会维护。所有引擎实现都以它为准。适合深入理解语言底层机制，但阅读门槛极高，通常不作为日常学习材料。

---

## 二、经典书籍

| 书名 | 适合阶段 | 说明 |
| --- | --- | --- |
| **《JavaScript 高级程序设计》**（红宝书） | 入门→中级 | 公认最全面的系统教程，覆盖语言核心 + DOM + BOM，最新第4版基于 ES6+ |
| **《你不知道的 JavaScript》**（YDKJS） | 中级→高级 | Kyle Simpson 著，深入作用域、闭包、原型链、异步等核心机制，免费开源 |
| **《JavaScript 语言精髓》**（蝴蝶书） | 中级 | Douglas Crockford 著，篇幅精炼，聚焦语言精华与糟粕 |
| **《ECMAScript 6 入门》** | 中级 | 阮一峰著，中文社区最广泛引用的 ES6+ 参考，免费在线阅读 |

---

## 三、权威在线资源

- **MDN Web Docs**（Mozilla） https://developer.mozilla.org/zh-CN/docs/Web/JavaScript 被称为 Web 开发的"百科全书"，API 参考文档的事实标准。所有浏览器厂商都参考和贡献 MDN。

- **TC39 提案仓库** https://github.com/tc39/proposals 追踪 JavaScript 未来特性的第一手来源。

- **JavaScript.info**（现代 JS 教程） https://zh.javascript.info/ 从基础到高级的系统化教程，讲解清晰，配有大量图示和交互示例。

---

## 四、实践与深入

- **V8 博客** — 了解引擎如何执行 JS
- **Can I Use** — 查看 API 的浏览器兼容性
- **TypeScript 官方文档** — 如果你在用 TS，这是必读

---

## 如果只选一个

> **入门** → 《红宝书》 + MDN **进阶** → 《你不知道的 JavaScript》 **查文档** → MDN（没有之一）

MDN 是日常开发中最频繁使用的权威参考，几乎没有替代品。书籍则提供系统化的知识框架，两者配合效果最好。
