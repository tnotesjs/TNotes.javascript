## 3. 📒 if...else 基本结构

- `if` 代码块后面，还可以跟一个 `else` 代码块，表示不满足条件时，所要执行的代码。
- 对同一个变量进行多次判断时，多个 `if...else` 语句可以连写在一起。
- `else` 代码块总是与离自己最近的那个 `if` 语句配对。

::: code-group

```javascript [if...else 基本结构]
if (x === 3) {
  // 满足条件时，执行的语句
} else {
  // 不满足条件时，执行的语句
}
// 上面代码判断变量m是否等于3，如果等于就执行if代码块，否则执行else代码块。

if (x === 0) {
  // ...
} else if (x === 1) {
  // ...
} else if (x === 2) {
  // ...
} else {
  // ...
}
```

:::
